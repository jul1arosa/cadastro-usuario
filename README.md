# 🚀 Cadastro de Usuários com Spring Boot

Este projeto é uma API simples para gerenciar usuários, focada em boas práticas de arquitetura e aprendizado do ecossistema Spring.

### 🔧 Tecnologias usadas:
* Java 24
* Spring Boot
* H2 Database (Banco em memória)
* Spring Data JPA
* Lombok
* Maven

### 📚 O que este projeto cobre:
* Estruturação de projetos Spring Boot (Controller, Service, Repository, Entity).
* Operações de CRUD.
* Configuração e uso do banco de dados H2.
